## Description

Astro Baltic Isikukood Verification

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Using the API

Submit a raw body POST request to the path `/verify` using JSON format.

Example: 

```
{
    "isikukood": "<INSERT YOUR ISIKUKOOD HERE>"
}
```
